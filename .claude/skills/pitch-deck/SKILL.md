# Pitch Deck

## Workflow
1. Get the company basics: problem, solution, stage, traction/metrics, ask (amount + use of funds).
2. Standard structure (adapt order to what's strongest first): Problem -> Solution -> Market size (TAM/SAM/SOM) -> Product -> Traction -> Business model -> Competition -> Team -> Financials/ask.
3. Each slide: one core idea, minimal text, a specific number or proof point where possible rather than vague claims.
4. Don't fabricate traction numbers, market sizes, or team credentials - flag clearly wherever the user needs to supply real figures.
5. Once the narrative/outline is approved, use the pptx skill to build the actual slide file.
