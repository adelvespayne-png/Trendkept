# Copywriting

## Workflow
1. Get the existing copy (or feature list) and the goal (clarity, conversion, tone shift, length).
2. Rewrite following: benefit before feature, active voice, one idea per sentence, concrete over abstract ("saves 3 hours a week" beats "boosts productivity").
3. Keep the user's actual claims - don't invent stats or features they didn't mention.
4. Offer before/after side by side so the change is easy to evaluate, and note why each change was made when it's not obvious.
