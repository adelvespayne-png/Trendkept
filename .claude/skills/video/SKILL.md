# Video

## Workflow
1. Get the video's goal, length, and platform (long-form YouTube vs 15-60s short).
2. Script format: Hook (first 2-3 sec, must be specific/visual) -> Setup -> Payoff/Value -> CTA. Include shot/visual notes alongside the spoken line, not just dialogue.
3. For shorts, write beat-by-beat with timestamps (0-3s, 3-8s, etc).
4. If actual video generation/editing is requested, note current tool limitations honestly rather than promising a rendered video this tool can't produce - offer the script + shot list as the deliverable, or check for an available image/video generation connector.
