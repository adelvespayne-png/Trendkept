# Ai Seo

## Workflow
1. Explain the shift: AI answer engines favor clear, quotable, well-structured factual statements, strong original data/stats, and clean semantic HTML (defined terms, direct Q&A framing) over keyword density.
2. Audit/rewrite content to:
   - Front-load direct answers to likely questions (answer in the first sentence of a section)
   - Include specific, citable facts/numbers/data the model can lift
   - Use clear headers phrased as questions where natural
   - Ensure the page is crawlable by AI bots (check robots.txt doesn't block common AI crawlers, if the user wants that)
3. This is a fast-evolving space - search for current best practice/guidance before giving specifics, since tactics here change often.
