# Social

## Workflow
1. Get the core message/topic and which platforms are needed.
2. Adapt per platform, don't just reformat the same text:
   - X: punchy, single idea, thread if it needs unpacking
   - LinkedIn: slightly longer, professional framing, personal angle or lesson
   - Instagram: caption + hook line for the image/video, hashtag suggestions
   - TikTok: hook (first 2 seconds), beat-by-beat script, not just a caption
3. Match the user's established voice if they've shared examples; otherwise ask for 1-2 reference posts.
4. Avoid generic hooks ("Let's talk about...", "In today's world..."). Lead with the specific, surprising, or concrete.
