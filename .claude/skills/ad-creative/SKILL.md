# Ad Creative

## Workflow
1. Get the offer, target audience, platform (Meta/Google/TikTok/LinkedIn - each has different copy norms and length limits), and any brand voice constraints.
2. Generate variants across different angles, not just rewordings of one angle: pain-point-led, benefit-led, social-proof-led, curiosity-led, urgency-led (if truthful).
3. Respect real platform character limits (e.g. Meta primary text ~125 chars for full visibility, headline ~40 chars).
4. Present as a table: angle | headline | primary text | CTA - so the user can scan and pick winners to test.
5. Never fabricate stats, reviews, or claims in the copy - flag anywhere the user needs to supply real proof.
