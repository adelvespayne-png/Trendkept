# Email Sequences

## Workflow
1. Identify the sequence type and trigger event (signup, cart abandonment, X days inactive, etc), and number of emails wanted.
2. Map the arc across emails - each should have one clear job (welcome/deliver value, handle objection, urgency/close, etc.) rather than repeating the same pitch.
3. Include for each email: subject line (with one alt to test), send-delay from trigger, and body copy.
4. Keep CTAs singular per email. Flag if the sequence relies on fabricated urgency/discounts - recommend only using ones the user can actually honor.
