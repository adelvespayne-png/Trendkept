# Programmatic Seo

## Workflow
1. Identify the data axis (locations, product variants, comparisons, integrations, etc.) and confirm the user has or can get a clean dataset (CSV/spreadsheet).
2. Design the page template: what's unique per page (data-driven facts, not just find-replace), what's shared (nav, boilerplate).
3. Flag thin-content/duplicate-content risk explicitly - programmatic pages that don't add unique value get deindexed. Push for at least one genuinely unique data point per page.
4. Provide: URL structure, template outline (sections + which are dynamic), and a sample of 2-3 filled-in pages.
5. If the user has the dataset uploaded, generate real filled templates rather than placeholders.
