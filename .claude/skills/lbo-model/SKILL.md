# Lbo Model

## Workflow
1. Gather: purchase price/multiple, financing structure (debt tranches, leverage ratio, interest rates), operating assumptions (revenue/EBITDA growth, margin), exit multiple and hold period.
2. Build sources & uses of funds at entry.
3. Project EBITDA growth and debt paydown (cash sweep) over the hold period.
4. Calculate exit equity value (exit EBITDA x exit multiple, less remaining net debt) and resulting IRR/MOIC to the sponsor.
5. Build as a spreadsheet with a clear sources & uses, debt schedule, and returns summary; include a sensitivity table on entry/exit multiple.
6. Frame as an educational/modeling exercise, not investment advice.
